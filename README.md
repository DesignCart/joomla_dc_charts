DC Charts
==========

DC Charts is a lightweight Joomla content plugin for Joomla 4 and 5 that lets you insert interactive charts directly into your articles using simple shortcodes. 
It supports pie, doughnut, bar, and line charts — no coding required.

Features:
- Four chart types: pie, doughnut, bar, line
- Fully responsive and mobile friendly
- No jQuery or external APIs — uses only Chart.js via CDN
- Custom titles, colors, legends, and axis labels
- Works with any Joomla template (including Bootstrap 5)
- Extremely lightweight (<10 KB)

Example usage:
{dcchart type="pie" title="Payment methods" labels="BLIK,Bank transfer,Card,PayPal,BNPL" values="38,32,15,8,7"}

Installation:
1. Download the ZIP package: https://www.designcart.pl/download/plg_content_dcchart.zip
2. In Joomla admin panel go to Extensions → Install → Upload Package File
3. Upload the ZIP
4. Enable the plugin under Content → DC Charts
5. Insert {dcchart ...} shortcodes into your articles
